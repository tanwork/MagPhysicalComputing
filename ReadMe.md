# Physical Computing for Passive Wireless Information Retrieval

- Author: Long Tan, Xiuzhen Guo, 
- Contact: {tanlongwork, guoxiuzhen94}@gmail.com, 
- Description: This repo hosts the software for the implementation of speech enhancement in the receiver side.

## File Structure
* Software includes the files needed for the  speech enhancement of the received encoding signals.
  * train.py: Training for the denoising model.
    
  * EvaluationTest.py: Evaluation for the denoising model.
    
  * \outputs\exp_: The results and weight file.
